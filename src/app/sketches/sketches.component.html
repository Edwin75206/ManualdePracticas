1. Landing Page

Página principal que se mostrará a todos los usuarios al entrar a la página, capta la atención de los usuarios y los anima a contratar algun tipo de servicio.

2. Servicio in Situ

Esta página mostrará el tipo de servicio y dirección de donde se hará el servicio, es decir será en el lugar que el usuario desee.

3. Servicio es sucursal

En este apartado se solicita un tipo de servicio y elige el lugar a donde irá, es decir el usuario escogerá el lugar donde tomará el servicio.

4. Ayuda

Para esta página se le ofrece al usuario o epresa una serie de opciones con las que cuenta para pedir ayuda a Agéndalo.

5. Seguridad

Esta página se encarga de exponer las diferentes medidas de seguridad que se toman para resguardar la información obtenida en esta plataforma.

6. ¿Quiénes somos?

En este apartado se explica lo que es Agéndalo, de donde surge, lo que esta comprometido a hacer, misión, visión y todo lo que la hace única.
